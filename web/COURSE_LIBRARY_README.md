# 课程库功能说明

## 功能概述
在Home页面的右上角按钮区域新增了"课程库"按钮，点击后会跳转到一个新的课程库页面。

## 功能特性

### 1. 课程库页面
- **页面路径**: `/course-library`
- **页面标题**: "课程库"
- **布局**: 类似主页的卡片网格布局

### 2. 课程卡片
- 显示课程名称、描述、时长、类型和创建日期
- 支持编辑和删除操作
- 美观的渐变背景设计
- 响应式布局，适配移动端

### 3. 添加课程
- 点击"添加课程"按钮打开弹窗
- 支持填写以下信息：
  - 课程名称（必填）
  - 课程描述
  - 课程时长（小时）
  - 课程类型（技术培训、管理培训、软技能培训、认证培训、其他）

### 4. 编辑课程
- 点击课程卡片上的编辑按钮
- 在弹窗中修改课程信息
- 保持原有的创建日期不变

### 5. 删除课程
- 点击课程卡片上的删除按钮
- 确认后永久删除课程

### 6. 数据存储
- 使用localStorage存储课程数据
- 数据键名：`course_library`
- 页面刷新后数据保持不变

## 文件结构

```
web/src/
├── views/
│   └── CourseLibrary.vue          # 课程库页面
├── components/
│   └── CourseCardSimple.vue       # 课程卡片组件
└── router.js                      # 路由配置（已更新）
```

## 使用方法

1. 在Home页面点击右上角的"课程库"按钮
2. 进入课程库页面后可以：
   - 查看所有已添加的课程
   - 点击"添加课程"创建新课程
   - 点击课程卡片上的编辑按钮修改课程
   - 点击课程卡片上的删除按钮删除课程
   - 点击"返回"按钮回到主页

## 技术实现

- 使用Vue 3 Composition API
- 响应式数据管理
- 本地存储持久化
- 组件化设计
- 现代化的UI设计
